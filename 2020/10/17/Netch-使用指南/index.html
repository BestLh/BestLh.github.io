Netch 使用指南

发表于 2020-10-17 更新于 2020-10-21

Netch 是一款运行在 Windows 系统上的开源游戏加速工具，简单易上手。也可以用于日常的网页浏览等。

Windows - Netch 使用教程 | 使用文档

简介

Netch 是一款运行在 Windows 系统上的开源游戏加速工具，简单易上手。也可以用于日常的网页浏览等。

Github 项目地址：https://github.com/NetchX/Netch

  • 不同于SSTap那样需要通过添加规则来实现黑名单代理，Netch原理更类似Sockscap64，通过扫描游戏目录获得需要代理的进程名进行代理。 也可以实现 SSTap 那样的全局 TUN/TAP 代理，和 shadowsocks-windows 那样的本地 Socks5，HTTP 和系统代理。
  • 在日常网页浏览方面，可以进行分流设置。
  • 支持的代理协议：Socks5 / Shadowsocks / ShadowsocksR / Vmess/Trojan(1.4.7版本后支持)
  • UDP NAT FullCone
  • 指定进程加速

整理教程时的系统环境：

Windows 10 1909

Netch 1.4.8

此文档适用于：跑路云/闪电

文档中的某些内容可能随时间变化而失效。

下载安装

  • 此软件为免安装版本，解压后即可运行
  • 下载地址（Trojan需下载1.4.7及之后的版本)https://github.com/NetchX/Netch/releases
  • Windows 64 位系统使用 x64 版本
  • Windows 32 位系统使用 x86 版本（1.4.3及之前版本才有32位)
  • 否则你会遇到驱动问题
  • 若使用 Chrome 浏览器下载 exe 文件，可能会提示不安全，请忽略此提示。

请注意：

此软件运行需要 .NET Framework 4.8 支持，

非 Windows 10 系统可能需要点此进行安装

快速上手

没有购买的推荐这里购买跑路云

下载压缩包后进行完整解压，并双击运行主程序 Netch.exe 。

Netch-01.png

添加服务器的方式主要有三种：

  1. 复制节点链接后从剪贴板导入。
  2. 手动填写服务器配置。
  3. 从订阅链接导入。

这里仅对V2ray订阅做出说明。

跑路云/闪电用户请在官网用户中心快速导入区域复制 V2ray订阅地址。

Netch-02.png

点击订阅，选择管理订阅连接。

Netch-03.png

在对应的地方粘贴自己的订阅链接，备注可以自行填写。点击添加后再点击保存。初次配置时不建议勾选左下角的选项，更新出节点后再来勾上（简单点说就是服务器列表没有节点不要钩）。

Netch-04.png

添加订阅后，需要回到主界面选择更新订阅，将服务器信息同步到本地。（更新不出来的看下面的更多疑问）

Netch-05.png

在节点列表选择一个节点。节点后方的数字为延迟。(经常抽风延时红的却能用)。

Netch-06.png

选择自己需要的代理模式。这里推荐 [4][网页代理] 绕过局域网和中国大陆 / Bypass LAN and China （低版本选这个，这里就不多做介绍），此模式下可以进行自动分流，不影响国内服务速度。

Netch-07.png

点击启动。

此时您的电脑已经可以访问国际网络。

为自己的游戏添加进程代理

如果你的游戏的模式已经被收录，也可以考虑在模式菜单中，选择使用已收录的模式。所有模式的文件，都在 ./mode/ 文件夹下，如果你需要多个模式的合并文件，可以使用记事本将其打开，将多个文件合并。

ping 值未必准确，因为这只是你本地到代理服务器而非游戏服务器的延迟。

如果你的游戏的模式没被收录，可以看接下来的扫描步骤来手动创建模式。

请点击菜单栏上的 模式 ，然后点击 创建进程模式。

右键自己的游戏或软件，打开文件所在的位置，复制文件地址。

由于编写教程时的电脑上没有安装游戏，故此处用 Chome 浏览器来演示，原理是相同的。

此时已经可以在代理模式窗口看到刚添加的进程模式了。

  • 启动 Netch 后，再去游戏根目录或者别的启动器如 Steam，Uplay 启动游戏即可。此时游戏就已经被代理了
  • 如果在 Netch 启动前就启动了游戏，建议重启游戏。
  • 如果需要 Steam，Uplay 等启动器也被代理，参照前面的方式对 Steam，Uplay 根目录也进行扫描即可
  • 如果出现了启动失败，或者无法代理成功的情况，请先尝试 选项 - 重启服务或选项 - 卸载服务，或者在退出 Netch 以后，点击运行在 Netch 根目录下的 DriverUpdater.exe 程序进行驱动更新。

常见疑问

1.订阅更新不出来？

前往网站节点列表选择一个可用节点复制节点链接，然后软件服务器从剪贴板导入，然后去管理订阅左下角勾上（此时有了需要勾上）,然后去更新就可以了。

Netch-19.png

Netch-20.png

2.没有网/全炸？

首先全炸除非跑路，否则肯定是你的问题，这里给出几个常见问题和解决方法

  • 校准时间，时间差距大可能有影响，可以去这里校准时间；
  • 默认模式可能没有代理你的浏览器，按上面游戏添加模式的方式给自己浏览器加一个模式；
  • 你本地网络炸了，换个好的宽带；
